CRUD Update & Delete (MERN Stack)

## What I did
Implemented PUT, PATCH, and DELETE REST API endpoints using Express.js and Mongoose.

## Tech Stack
- Node.js, Express.js, Mongoose, MongoDB Atlas

## Endpoints
| Method | URL | Description |
|--------|-----|-------------|
| PUT    | /api/users/:id | Full update |
| PATCH  | /api/users/:id | Partial update |
| DELETE | /api/users/:id | Delete user |

## How to run
1. Clone repo
2. `npm install`
3. Add `.env` with MONGO_URI
4. `node server.js`
